# The Dog Park Finder – v2.04

Static frontend for dog-park discovery across Europe.

## Structure

index.html
pricing.html
advertisers.html
partners.html
css/
js/
data/

Uses `parks.json` as data source.

## Deployment

### GitHub Pages

1. Create repo `thedogparkfinder`
2. Upload all files
3. Settings → Pages → Deploy from branch → `main` → root

Site opens at:

`https://aoblak.github.io/thedogparkfinder`

### Custom domain

Add:

- `A` record → GitHub Pages IPs  
- `CNAME` file with your domain  
- Verify in Pages settings  

### Vercel / Cloudflare Pages

Import repo, no config needed.
